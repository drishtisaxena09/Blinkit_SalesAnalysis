# 🛒 Blinkit Sales Analysis

This project performs Exploratory Data Analysis (EDA) on sales data from **Blinkit** using Python. The goal is to extract meaningful business insights by analyzing product performance, outlet characteristics, and sales distribution.

---

## 📌 Project Objective

* Clean and preprocess raw sales data
* Perform exploratory data analysis (EDA)
* Visualize sales trends and patterns
* Generate business insights for decision-making

---

## 🛠️ Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📊 Key Analysis Performed

* Handling missing values
* Data type corrections
* Sales distribution analysis
* Product category performance
* Outlet size & location impact on sales
* Visual insights using graphs and charts

---

## 📈 Key Insights

* Identified high-performing product categories
* Analyzed how outlet type and size affect revenue
* Discovered patterns in overall sales distribution

---

## 🚀 Skills Demonstrated

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Business Insight Generation
